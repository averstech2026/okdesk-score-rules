# Открытые вопросы

Заполнить и утвердить, прежде чем внедрять в Okdesk и премию.

## Каталог

- [ ] Полный список базовых категорий (достаточно ли T5 / C15 / S30 или нужны ещё коды).
- [ ] 5–10 эталонных примеров на каждую категорию.
- [ ] Порог часов для `X_LONG` и финальные значения надбавок.
- [ ] Потолок суммы надбавок на заявку.

## Okdesk

- [ ] Какое поле(я): категория, осложнение, причина выезда (обязательность).
- [ ] Как отличить конвейерных клиентов/очереди от общего пула (список).
- [ ] Есть ли надёжные метки: время взятия в работу vs создание в группе.
- [ ] Как стыковать календарь дежурств с отчётом (без ручного ×2).

## Выезды

- [ ] Точное определение границ Мск / ближнее / дальнее.
- [ ] Нужно ли всегда отдельно бить «зона + работы» или бывают пакетные исключения.

## Организация

- [ ] Кто владелец пула на смену и как назначается.
- [ ] Правила взятия из групповой очереди.
- [ ] Кто проводит выборочный аудит и сколько часов в месяц.

## Премия

- [ ] Веса: только баллы по сегментам vs баллы + качество.
- [ ] Как смешивать человека, который часть месяца на конвейере, часть в пуле.
- [ ] Связь дежурного отчёта (1,5 недели) с месячной премией — без двойного счёта.
