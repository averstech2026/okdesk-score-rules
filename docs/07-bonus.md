# Формула премии (черновик)

Цель: сохранить баллы как основу, убрать стимул «только конвейер и самооценка».

## Контуры отчётности (как сейчас)

| Контур | Период | Основа |
|--------|--------|--------|
| Основная премия | Месяц | Баллы по правилам + корректировки |
| Дежурства | ~1,5 недели | Баллы дежурных смен (уже с ×2) |

## Предлагаемая логика v1 (к утверждению)

```
премиальная_база = нормативные_баллы_за_период
                   (по справочнику + осложнения + выезды, ×2 только в дежурстве)

премия ≈ f(
  премиальная_база в контексте сегмента,
  качество / персональный SLA-после-взятия,
  результат выборочного аудита
)
```

### Вариант интерпретации (не числа, а смысл)

1. **Объём** — нормативные баллы, но отчёт и сравнение **по сегментам** (`conveyor` / `pool`), либо относительно медианы сегмента.
2. **Качество** — небольшой вес (например корректирующий коэффициент 0.9–1.1), когда появятся стабильные данные.
3. **Аудит** — срезание/пересчёт по заявкам с неподтверждённой категорией или надбавкой.

Конкретные проценты весов и пороги рублей/баллов — **не зафиксированы**, см. open-questions.

## Чего не делаем в v1

- Не отменяем баллы в пользу «чистого KPI».
- Не вешаем групповой SLA на личную премию.
- Не вводим отдельный множитель внеурочки вне дежурства.
