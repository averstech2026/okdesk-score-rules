# Дежурства

## Правило

В **дежурную смену** по утверждённому графику итоговые баллы заявки (база + осложнения + выезд), закрытой/выполненной в рамках этой смены, умножаются на **2**.

```
итого = (база + осложнения + выезд) × 2   // дежурство
итого = (база + осложнения + выезд) × 1   // обычное время
```

## Источник истины

Коэффициент ставит **не инженер**, а правило на основе календаря/табеля дежурств.

## Вне дежурства

Работы в нерабочее время **вне своей дежурной смены** (массовые обновления, раскатки и т.п.) пока **без** отдельного коэффициента — отдельной боли нет. При появлении системной несправедливости — вынести в отдельную статью, не смешивая с дежурством.

## Отчётный цикл

Дежурные смены обычно закрываются отдельным отчётом с циклом порядка **1,5 недели** (как в текущей практике). Обычные баллы — в месячном контуре премии.
