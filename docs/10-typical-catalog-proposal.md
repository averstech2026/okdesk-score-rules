# Ревизия справочника «Типовая проблема»

Связано с аналитической страницей (Cursor Canvas) и выгрузкой за 23.01–22.07.2026.

См. также [11-ticket-contours.md](11-ticket-contours.md): часть «Другое»/пусто — это **массовый перенос из хелпдеска MFC** и шаблонное техзакрытие, а не отказ инженера выбрать пункт.

## Диагноз

| Контур | Заявок | Другое + пусто |
|--------|-------:|---------------:|
| Вся выборка | 19 640 | 37% |
| Только MFC | 6 574 | **67%** |
| Без MFC (нативнее) | 13 066 | **22%** |

- Комментарий «Техническое закрытие» — про процесс переноса/архивации, **не** про качество справочника.
- Каталог и KPI заполнения `typical` считать **без MFC-bulk**.
- Без MFC лидер — «Доп. настройки…» (catch-all); «Другое» всё ещё ~1 250 — там искать пробелы (НДС, раскатки, карты/клиенты, сервисы вроде Work Bright).
- «Не обновились лимиты» (19) — кандидат на слияние в «Карты и клиенты».

## Предлагаемые объединения

| Было | Станет |
|------|--------|
| Оплата ДУЭТ + банк/процессинг | Оплата не проходит (банк / процессинг / ДУЭТ) |
| Вопросы по отчётам + не печатается отчёт (+ сверки) | Отчёты и расхождения данных |
| Карта клиента + лимиты + «клиенты» из нативного Другое | Карты питания и клиенты |
| Отменить чек | Корректировка чека (сторно, возврат, удаление, перенос) |
| Консультация | Консультация / звонок |
| Не работает ЧЗ | ЧЗ / маркировка |

## Предлагаемые добавления

1. **НДС / фискальные ставки**
2. **Массовое обновление / раскатка ПО** (спулер, SDK, лого…)
3. **Запуск / подготовка** (рядом с подключением оборудования)
4. **Сопутствующий сервис клиента** (Work Bright / AllBusiness и аналоги)

## Что внутри двух корзин (кратко)

### «Другое» (3639; без MFC 1248)
Карты/клиенты, звонки, чеки/удаления, расхождения, обмены, Work Bright/AllBusiness, НДС, письма по ЧЗ, хвост нераспознанного.

### «Доп. настройки, сопутствующий сервис…» (2203)
Не то же, что Work Bright. Внутри: карты/дотации (~464), обмены/меню (~221), доступы (~122), раскатки ПО (~71), код кассира/портал, ЧЗ/ПИОТ, запуски. Пункт нужно **сузить**; внешние сервисы клиента — отдельной строкой.
