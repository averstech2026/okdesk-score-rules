Легенда расчёта

  • 1 балл = 15 ₽. Премия (₽) = сумма баллов × 15.
  • Две статьи расходов по чекбоксу watch (Дежурство): обычные заявки без дежурства и дежурство заявки с дежурством — считаются и сравниваются отдельно.
  • Текущий — поле ticket_weight как проставил инженер (нет баллов → 0 ₽).
  • Новый (scored) — нормативный каталог только там, где баллы уже стояли.
  • Новый (все) — нормы на заявки с работой; MFC-техbulk без баллов = 0; компенсатор MFC = T5×N за день.
  • Формула нормы: без выезда (база × N + осложнение); с выездом только 60; в дежурстве ещё ×2.
  • Базы: T5=5, C15=15, S30=30; выезд = только +60 (без суммы с typical); осложнение +15/+30 только без выезда.
